# IT Support Lab - Brisbane

Practical project simulating a small business IT support environment.
Configured on Ubuntu Server running on VirtualBox.

## Objective

Build hands-on IT Help Desk skills by managing real services,
users, and troubleshooting incidents.

## Server Architecture

- **Apache2** - internal web server
- **Samba** - shared file server
- **MySQL + PHP** - database and backend for osTicket
- **osTicket** - ticket management system
